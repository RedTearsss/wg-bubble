Wg-bubble

给你的 WordPress 加个彩蛋，模拟 Windows 气泡屏幕保护，显示最近评论数量最多的访客头像，支持 IE9 / Chrome / Firefox / Opera 等浏览器，可配置等待时间、气泡半径以及气泡数量等。

距离上一款万戈牌 WordPress 插件已经整整过去一年了，这一年来，因为忙于工作，很少有折腾 WordPress，万戈牌没落了，但是并没有消失，今天万戈又给大家带来一款全新的 WordPress 插件 —— 气泡屏保插件！

WordPress 气泡屏保插件是什么？

是不是厌倦了你那平淡无奇的 WordPress？想不想给你的博客加一个彩蛋？想不想给你的访客一个 surprise？想不想让你的网站有点与众不同？只要你愿意，轻点鼠标启用万戈牌 WordPress 气泡屏保插件就可以做到这些。正如这个插件的名字，它和 Windows 的气泡屏保有着异曲同工的作用，当你的鼠标停止移动，一段时间后在浏览器的左下角就会一个个地冒出五彩的半透明气泡来，这些气泡会在浏览器中飞舞碰撞，直到你的鼠标再次开始移动，这些气泡就立马消失了。

WordPress 气泡屏保插件怎么用？

为了让这个插件最简单易用，你需要做的只是下载并启用万戈牌 WordPress 气泡屏保插件就可以了，然后访问你的 WordPress，鼠标停止移动1分钟以后，就可以看到效果啦！当然，为了满足更多个性化地需求，我也给这个插件加上了后台控制，可以设置的参数选项有等待时间（1 ~ 10分钟）、气泡半径（10 ~ 100像素）、气泡数量（1 ~ 15个）。

WordPress 气泡屏保插件说明

1、因为插件中的气泡动画是用 canvas + javascript 绘的，所以 ie6/7/8 不支持，但是使用本插件不会对 ie6/7/8 的访问有任何影响。

2、气泡的质感效果和 Windows 的屏保仍有差距，本来想加上高光、径向渐变、阴影等效果，可惜这些都太耗内存，部分浏览器下效果卡顿，无奈只好放弃。

3、和其他的万戈牌系列插件一样，我稍后还将另外提供一个非插件的气泡屏保版本，可供移植到其他博客平台或者网站，敬请关注。

插件更新记录

2013.04.21

在插件发布之后，感谢各位同学提出的意见，有的说气泡太难看，有的说把气泡换成 trollface，有的说改成评论排行的头像，我汇总了一下，大多数的意见是觉得气泡效果不好，所以我就索性把这个插件改成了另类的“读者墙”，显示的不再是气泡，而是近期在 WordPress 发表评论最多的几位访客头像和名字，以及相应的链接。众口难调，只能先做成这样了。
